Sobre nós

A Comunidade JAVA Ribeirão Preto tem como objetivo promover o desenvolvimento e a troca de conhecimentos sobre a linguagem Java. Oferecemos eventos, workshops e networking para todos os interessados em Java.

Eventos

Encontro Mensal de Desenvolvedores

Discutir temas relevantes à área, trocar experiências, fazer networking e aprimorar suas habilidades

Workshop de Introdução ao Java

Visa fornecer aos participantes os fundamentos da linguagem de programação Java, preparando-os para desenvolver aplicações básicas e compreender os conceitos essenciais da orientação a objetos

Palestra sobre Java 21

Vamos abranger os novos recursos, melhorias de desempenho e mudanças na plataforma que a versão 21 do Java oferece
